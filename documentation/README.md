# HOW TO USE

## NEW TAGS :

```HTML
<any-></any->
```

> example :

```HTML
<flex-></flex->
```

## NEW Attributes

```HTML
<any- attrib ></any->
<any- attrib="any" ></any->
```

> example :

```HTML
<flex- g6 ></any->
<flex- align-items="center" ></any->
```
